# 📝 MERN Todo List App

A full-stack **Todo List application** built using the **MERN stack (MongoDB, Express, React, Node.js)**.  
This is my **full-stack solo project**, I learned how the frontend and backend communicate through APIs and how basic **CRUD operations** work in a real-world application.

---

## 🚀 Features
- Create, read, update, and delete todos
- RESTful API for frontend–backend communication
- Persistent data storage with MongoDB
- Responsive UI styled with Tailwind CSS
- Reusable and consistent color system using predefined CSS variables
---

## 🛠️ Tech Stack
### Frontend
- React
- Tailwind CSS
- CSS variables for theme colors
- Axios / Fetch API

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

---

## 📚 What I Learned

This project helped me understand key full-stack concepts, including:

- How **frontend and backend communicate using REST APIs**
- Handling **HTTP requests** (GET, POST, PUT, DELETE)
- Implementing **CRUD functionality**
- Structuring a backend server with Express
- Connecting and working with **MongoDB** through **Mongoose**
- Managing state and side effects in React
- Making asynchronous API calls
- Styling a React app efficiently using Tailwind CSS
- Organizing and using reusable color variables

---

## 🔄 CRUD Operations

- **Create** – Add a new todo to the database
- **Read** – Fetch and display todos from the backend
- **Update** – Edit an existing todo
- **Delete** – Remove a todo from the database
